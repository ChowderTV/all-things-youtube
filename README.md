# Elojire — YouTube Resource Hub

Welcome! This repo is the companion to my [YouTube channel](https://www.youtube.com/@Elojire) — a place to grab everything you see in my videos: source code, Claude skill files, prompt files, and project templates.

---

## 📁 What's Inside

| Folder | Description |
|--------|-------------|
| `tutorials/` | Full source code from coding tutorials (React, TypeScript, Python, CSS) |
| `skills/` | Claude skill files (`.md`) featured in videos |
| `prompts/` | Prompt files and system prompts used or discussed on the channel |
| `templates/` | Starter templates and boilerplate you can clone and use |

> Each folder contains its own `README.md` with context for what's inside and which video it came from.

---

## 🎬 How to Use This Repo

1. **Find the video** — each folder or file is named to match the video title or topic.
2. **Clone or download** what you need:
   ```bash
   git clone https://github.com/Elojire/<repo-name>.git
   ```
3. **Follow along** — instructions and setup steps are in each project's own README.

---

## 🤖 Claude Files

The `skills/` and `prompts/` folders contain files designed to work with [Claude by Anthropic](https://claude.ai). To use them:

- **Skill files** — place them in your Claude skills directory as shown in the video.
- **Prompt files** — copy the contents into Claude's system prompt or paste directly into the chat.
